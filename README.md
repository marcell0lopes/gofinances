## **Stack**

- ReactNative
- ReactHookForm
- React Native Gesture Handler
- React Native Navigation
- React native SVG Transformer
- Styled Components
- Expo
- Expo Auth (Google e Apple OAuth2)
- Expo Splash Screen
- TypeScript
- Date-FNS
